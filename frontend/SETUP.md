# Vite + React Frontend

Este diretório contém a aplicação frontend para o **Consultor do Futebol Brasileiro**.

## 🚀 Quick Start

```bash
# Instalar
npm install

# Rodar localmente
npm run dev

# Build para produção
npm run build
```

## 📖 Documentação Completa

Veja [README.md](./README.md) para documentação completa, estrutura do projeto, guia de desenvolvimento, e mais.

## 🔗 API Backend

Conecta-se à API FastAPI rodando em `http://localhost:8000` por padrão.
Configure em `.env.local` com `VITE_API_URL`.

## ⚽ Funcionalidades

- Listagem de clubes com busca
- Detalhes de partidas (gols, cartões, estatísticas)
- Tabela de classificação por ano
- Rankings (artilheiros, cartões)
- Confronto direto entre clubes
- Interface responsiva com Tailwind CSS
